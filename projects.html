Gridlock network analysis
非定常時の経路選択行動分析

#GPS #BehaviorModeling #ModelEstimation #Routechoice
In extraordinary networks, such as hyper-congested networks, drivers do have little knowledge of networks due to a lack of experience or information. In such a context, drivers choose routes based on dynamic decision making, which is sequential and somewhat forward-looking, and this can make the network throughput even worse. In this project we propose a dynamic route choice model with a sequential discount rate, which can describe the dynamic decisions of drivers, while the previous models can model either global or myopic decisions. We estimated the model using taxi probe data in a gridlock network and analyzed the change of drivers' behavioral mechanisms.
超混雑等の非定常ネットワークでは，ドライバーの経路選択に関わる意思決定が近視眼的になり， その行動がネットワークの状況をさらに悪化させる可能性がある． 従前の経路選択分析は全体最適の仮定に基づくもので，こうした動学的な意思決定のメカニズムを記述できなかった． 本プロジェクトでは，割引率の概念を導入した動的逐次経路選択モデルを提案し， そのモデルを東日本大震災発生日を含む複数日・複数時間帯のタクシープローブデータに適用し， 災害時のドライバーの経路選択メカニズムの変化を明らかとした．

Route choice measurement
歩行者経路選択行動の観測

#GPS #MeasurementModel #ModelEstimation #Routechoice
Route choice behavior in real networks are usually measured by matching GPS location data with real maps. When measurement errors of data are large, in particular urban pedestrian networks where narrow streets or streets with arcades, the measurement uncertainty in the map-matching process leads to biases in the route choice model estimation. In this project, we propose 1) a link-based Bayesian route measurement model that estimates street-specific variances of measurement errors and 2) a structural estimation method that reduces the biases in parameter estimation.
経路選択モデルの推定に必要な，実際の経路選択行動の情報は通常，GPS等のデータをマップマッチング処理して観測される． しかしデータの観測誤差が大きい場合，特に路地やアーケードの存在する歩行者ネットワークでは，経路観測の不確実性が高く， それによって行動モデルの推定にバイアスが生じる． 本プロジェクトでは，観測誤差分散を街路固有の未知変数として扱うリンクベースのベイズ経路観測モデルと， 行動モデル・パラメータのバイアスを軽減する構造推定手法を提案した． これにより，観測誤差の影響で，従前では難しかった行動メカニズムの特定に成功した．
Article (EN)

Network design for pedestrian activities
歩行者活動ネットワークの最適化

#Optimization #NetworkAssignment #ActivityModeling
Network design is an optimization framework based on the prediction of travel demand and has been mainly applied in studies for the road traffic networks. This project extends the framework to "pedestrian activity network design" to consider the impact of design in terms of activities (i.e. time-use) as well as of travels. Using it, we formulated a pedestrian network design problem for the city center of Matsuyama-city, Japan, and showed the Pareto frontier solution set with different budgets.
ネットワーク最適化（Network design）は，人々の移動需要の予測を下敷きにしたネットワーク設計変数の決定問題であり， 主に道路交通ネットワークを対象に研究が進められてきた． 本プロジェクトでは，歩行者の街路設計変数が及ぼす影響を，移動のみならず活動（まちなかでの時間利用）の視点から考えるため， 「歩行者の活動ネットワーク」最適化問題へと枠組みを拡張した． 実際に，回遊行動への影響を考慮した松山市中心市街地における街路ネットワーク設計へ適用し，投入予算に応じた（パレート）最適解の集合を示した．

Activity assignment model
活動配分モデルの開発

#ActivityModeling #NetworkAssignment #Routechoice
Pedestrian sojourn activity is a complicated decision which includes the choices of activity places, durations and walking routes for multiple activities. This project proposes an "activity path" choice model with time-space constraints by extending a Markovian route choice model in a time-space network. This model can also be used for network assignment with which we can evaluate aggregate activity demand patterns in a network.
歩行者の回遊行動は，活動場所・活動時間，そして移動街路の選択がミクロなレベルで組み合わさった，複雑な意思決定の問題である． 本プロジェクトは，マルコフ型の経路選択モデルを時空間ネットワーク上へ展開し，時間制約下での「活動経路」選択モデルとして， それらの選択を統合的に記述した． さらに配分モデルへと拡張し，ネットワーク上の集合的な移動・活動分布を評価可能な枠組みを提示した．
Article (JP) Presentation (JP)

Need-based activity generation analysis
活動欲求に基づく活動発生分析

#BehaviorModeling #ModelEstimation #Psychology #ActivityModeling
This project incorporates a psychological mechanism called "activity needs" into a discrete-continuous model to describe the sequence of activity generations and duration choices. We analyzed the model using real pedestrian trajectory data in the city center of Matsuyama, Japan, focusing on the role of "small" activities and locations of urban facilities.
本プロジェクトは，従前研究手法では難しかった回遊行動の心理的メカニズムに対して， 「活動欲求」の概念を導入した離散-連続モデルを提示し，離散的な活動実行選択と 連続的な時間選択のシークエンスが互いに関連する回遊行動を記述することに成功した． 「小滞在」がもつ媒介活動としての意味に着目した実行動データの分析により， まちなかの交通・活動施設の配置が小滞在の活動発生確率と滞在時間に及ぼす影響を 定量的に明らかとした．

Pedestrian street choice analysis
歩行者の街路選択行動

#MachineLearning #BehaviorModeling #ModelEstimation #RouteChoice
This project first analyzes the streetscape pattern in Shibuya city center network by clustering approach based on street built-environment data. We then study the streetscape effect on pedestrian street choice behavior.
本プロジェクトでは，まず，多彩な街路環境変数から渋谷の街路景観パタンをクラスタリングし， さらに街路景観パタンが歩行者の逐次的な街路選択行動に及ぼす影響を定量的に分析した． その結果，鉤型に連続する景観パタンの類似した街路が選択されやすい傾向を明らかとした． なお，本プロジェクトは柿元淳子氏の2011年度東京大学修士論文「歩行者の移動軌跡に着目した都市街路網における経路選択機構の分析」に基づくものである．

Interaction in a shared space
共有空間における衝突回避・譲歩行動

#Pedestrian #Microsimulation #GameTheory #ModelEstimation
For designing "shared space"s, it is important to better understand the interactions between pedestrians and the other agents like vehicles. This project develops discrete choice models with game theory to describe the micro movements of pedestrians and of vehicles as well as their interactions.
共有空間（Shared space）の設計において，歩行者と，自動車等他の交通主体との相互作用を理解することは重要である． 本プロジェクトは，歩行者と自動車のミクロな動きとその相互作用を，ゲーム理論のメカニズムを組み込んだ離散選択モデルによって記述した． なお，本プロジェクトは北川直樹氏の2009年度東京大学修士論文「移動空間の更新に伴う歩行者と自動車の相互規範の変化」に基づくものである．
